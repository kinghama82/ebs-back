🎞프로젝트 개요
------------------
보드게임 정보검색 및 게임기록 작성을 통한 커뮤니티 웹서비스
<br/>

### 🎲주요기능 ##
검색툴을 이용한 게임정보 검색   <br/>
친구들과의 즐거웠던 게임기록 작성   <br/>
보드게임을 잘 모르는 사람들을 위한 게임추천  <br/>
다양한 보드게임 정보를 교환할수있는 게시판기능  <br/>
